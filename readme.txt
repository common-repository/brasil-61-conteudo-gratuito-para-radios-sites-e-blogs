Plugin para importação automática de notícias do portal Brasil61.

== Description ==
O Brasil 61 fornece diariamente conteúdos radiofônicos e em texto web para comunicadores de rádio e mídias digitais online de todo o Brasil. Todos os conteúdos são gratuitos, livres para veiculação, e tratam de temáticas regionalizadas e informações estratégicas para os municípios, como repasses de recursos federais para as prefeituras de todo o país.

Agora, você pode manter um site que se atualiza automaticamente, com novos conteúdos diários. A melhor parte? O plugin do Brasil 61 é gratuito e vai te ajudar a impulsionar seu marketing de conteúdo, a melhorar o seu SEO, a aumentar o envolvimento do seu público e trazer mais visualizações. Tudo isso pode trazer um maior ganho financeiro para seu veículo.

Experimente o plugin do Brasil 61 hoje mesmo!

== Installation ==
A instalação do plugin é simples. Basta acessar a aba "Plugins > Instalar novo" no seu painel administrativo do WordPress e buscar pelo plugin "Brasil 61" na barra de busca.

Ao ativar o plugin, será necessário uma chave de ativação para ter acesso ao conteúdo do Brasil 61 e configurar o plugin. Para isso, basta clicar no link "Obtenha a chave" no painel de ativação de conta, preencher o formulário e enviar.

Caso você já tenha uma chave, basta inserir e salvar!

Importante informar corretamente seus dados, pois é por meio deles que você receberá a chave. Para manter o plugin ativo, pedimos que mantenha seu cadastro no Brasil61.com atualizado, utilizando seu melhor e-mail e número de WhatsApp.


== Frequently Asked Questions ==
= Como configurar o plugin? =
É fácil e intuitivo! Você pode customizar suas importações, extrair todos os conteúdos de uma categoria do Brasil 61 para uma categoria do seu wordpress. Pode pegar apenas conteúdos de uma tag do Brasil 61 e levar para uma determinada categoria do seu worpress. 

Nas configurações, também é possível definir se você quer importar as imagens, resumo e criar backlink. O backlink é um link de volta para o artigo original - essa ação é importante para que possamos continuar melhorando a qualidade de entrega dos nossos conteúdos. Também, você pode definir o autor do artigo. 

Sugerimos que crie um usuário do tipo autor com o nome "Brasil 61". 
No item "Status", pode definir se o post será publicado automaticamente após a importação ou se irá para rascunho ou revisão pendente.

= Periodicidade da atualização da importação das notícias =
Você pode definir a periodicidade que seu WordPress irá até o brasil61.com buscar novos conteúdos - seja a cada 30 minutos, a cada hora, uma vez por dia, duas vezes por dia ou semanalmente. 

Além disso, pode você pode configurar a data inicial da importação, permitindo a importação de conteúdos mais antigos para popular seu blog ou site.
